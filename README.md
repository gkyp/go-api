how to run:  
	clone the repo  
	run ```go build```  
	```./ptlist``` to run with default port (8080)  
	```./ptlist [port number]``` to specify custom port, e.g. ```./ptlist 8081```  
	
	curl -v "http://localhost:8080/ptlist?period=1h&tz=Europe/Athens&t1=20210714T204603Z&t2=20210715T123456Z"
